# Projeto de Prática com HTML e CSS - Huddle Landing Page Responsiva

Este repositório contém um exercício prático desenvolvido para testar e reforçar conhecimentos em **HTML** e **CSS**, com foco em **responsividade** e **Flexbox**. O projeto recria a página de introdução do Huddle, seguindo o desafio proposto pelo Frontend Mentor.

## Objetivo
O objetivo principal deste projeto é aplicar regras de layout responsivo e Flexbox para criar uma landing page adaptada a diferentes tamanhos de tela, mantendo uma experiência de usuário consistente.

## Tecnologias Utilizadas
- **HTML5**: Estrutura semântica do conteúdo.
- **CSS3**: Estilização e responsividade com:
  - **Flexbox** para alinhamento e distribuição de elementos.
  - **Media Queries** para adaptação do layout em telas menores.
- **Google Fonts**: Poppins e Open Sans.
- **Font Awesome**: Ícones de redes sociais.

## Estrutura do Layout
- **Cabeçalho**: Exibe o logotipo.
- **Seção Principal**: Inclui uma ilustração, um título descritivo, um parágrafo explicativo e um botão de registro.
- **Rodapé**: Ícones de redes sociais estilizados.

## Estrutura do Projeto
- `index.html`: Estrutura HTML do layout responsivo.
- `reset.css`: Reset de estilos para uniformização do design.
- `estilos.css`: Estilos principais e aplicação de Flexbox.
- `responsivo.css`: Regras específicas para adaptação em diferentes tamanhos de tela.
- Pasta `src/images`: Contém imagens e ícones utilizados no projeto.
